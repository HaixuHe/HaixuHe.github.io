配置GitHub Token

如何获取GitHub Token：

  1. 访问 GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 点击 "Generate new token (classic)"
  3. 勾选 repo 权限（必须勾选此项）
  4. 生成并复制Token
可以只输入仓库名（如 HaixuHe.github.io），系统会自动获取你的用户名
或输入完整格式：用户名/仓库名（如 HaixuHe/HaixuHe.github.io）
